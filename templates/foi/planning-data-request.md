# Freedom of Information Request - Planning Application Data

**To:** [Council Name] Planning Department
**From:** NW London Local Ledger

## Request under the Freedom of Information Act 2000

Dear [Council Name],

I am writing to request information under the Freedom of Information Act 2000.

I would like to request the following planning application data for your council area:

### Data Requested

1. **Planning Applications Data** (last 12 months)
   - Application reference numbers
   - Application dates (received, validated, decided)
   - Application types (full, outline, householder, etc.)
   - Application status (pending, approved, refused, withdrawn)
   - Site addresses
   - Applicant names (where not personal/residential)
   - Agent details (where applicable)
   - Development descriptions
   - Decision dates
   - Officer recommendations
   - Committee decisions (where applicable)

2. **Format Requirements**
   - Preferred format: CSV or Excel spreadsheet
   - Alternative formats: JSON, XML, or database export
   - Please include column headers/field descriptions

3. **Additional Information**
   - If bulk data export is available via your planning system
   - If you have an API or data portal for planning data
   - Update frequency of your planning data

### Purpose of Request

This data will be used for:
- Providing free public access to planning information
- Supporting transparency in local development
- Helping residents understand local planning activity
- Non-commercial community information service

### Legal Basis

This request is made under:
- Freedom of Information Act 2000
- Environmental Information Regulations 2004 (where applicable)
- The Re-use of Public Sector Information Regulations 2015

### Cost and Delivery

I understand that you are not obliged to comply with this request if the cost exceeds the appropriate limit (£450/£600). If you estimate the cost will exceed this limit, please:

1. Advise how the request could be refined to bring it within the limit
2. Consider providing the information in stages
3. Advise if data is available through alternative means

Please provide the information electronically to: foi@nwlondonledger.com

### Timescale

I would appreciate a response within the statutory 20 working days. If you require clarification of any part of this request, please contact me as soon as possible.

If you are unable to provide any part of the requested information, please:
- Specify which exemptions you are relying upon
- Explain why the exemption applies
- Consider whether partial disclosure is possible
